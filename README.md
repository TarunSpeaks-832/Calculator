 Modern Web Calculator

A fully functional arithmetic calculator developed as a web application. This project demonstrates proficiency in **DOM manipulation**, **event handling**, and **responsive UI design**.

## Key Features
- **Basic Operations:** Perform addition, subtraction, multiplication, and division with precision.
- **Clear & Delete:** Integrated functionality to clear the entire display or delete the last character entered.
- **Responsive Layout:** Designed with **CSS Grid** to ensure a seamless experience across mobile and desktop devices.
- **Input Validation:** Prevents multiple decimal points and handles edge cases like division by zero.
